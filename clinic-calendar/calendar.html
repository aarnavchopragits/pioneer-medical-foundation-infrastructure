⚙️ Setup Required — Configure your Microsoft Outlook Calendar connection

Open this HTML file in a text editor and fill in the CONFIG block near the top of the <script> tag, then re-open the file.

Step 1 — Register an Azure AD app Go to portal.azure.com → Azure Active Directory → App registrations → New registration. Set the redirect URI to http://localhost (or your domain) as a Single-page app (SPA).
Step 2 — Grant calendar permissions Under your app → API Permissions → Add → Microsoft Graph → Delegated → add Calendars.Read. Grant admin consent if required.
Step 3 — Copy your Client ID & Tenant ID From the app's Overview page, copy the Application (client) ID and Directory (tenant) ID into the CONFIG below.
Step 4 — Find your Calendar ID Leave CALENDAR_ID blank to use the primary calendar, or use Microsoft Graph Explorer to call GET /me/calendars and copy the id of the shared calendar.
