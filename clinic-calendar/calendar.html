⚙️ Setup Required — Configure your Google Calendar connection

Open this HTML file in a text editor and update the CONFIG block near the top of the <script> tag.

Step 1 — Get a Google API Key Go to console.cloud.google.com → Create project → Enable Google Calendar API → Credentials → Create API Key. Restrict it to the Calendar API and your domain.
Step 2 — Make your Calendar public In Google Calendar, click the three dots next to your calendar → Settings and sharing → check Make available to public.
Step 3 — Find your Calendar ID In Calendar settings → Integrate calendar → copy the Calendar ID (looks like abc123@group.calendar.google.com).
Step 4 — Add RSVP links to events In the event description, add a line like RSVP: https://your-link.com and it will appear as a button on the calendar.
